# HRDD: Hadamard–ReLU Drift Detector

This repository contains the implementation and experimental materials for **HRDD**, an unsupervised concept drift detector based on Hadamard-transformed feature representations and statistical testing.

The method is designed for chunk-wise data stream analysis, where the goal is to detect changes in the data distribution without requiring class labels at the detection stage.

## Repository contents

The repository includes:

* implementation of the HRDD detector,
* scripts for generating synthetic data streams,
* scripts for running comparative experiments,
* detector configurations,
* result files,
* visualization scripts used to generate figures for the manuscript.

## Main idea

HRDD transforms incoming data chunks using a Hadamard-based projection followed by a ReLU activation. The transformed representation is compared with a reference window using repeated statistical tests. A drift alarm is raised when the number of significant changes exceeds a predefined voting threshold.

The detector is unsupervised, meaning it does not require true class labels to decide whether to raise a drift alarm.

## Experimental setting

The experiments evaluate whether detector alarms occur close to the true drift positions in synthetic data streams. This setting is important because only synthetic streams provide known ground-truth drift locations.

The evaluation uses the following criteria:

* **D1** — detection-centric distance,
* **D2** — event-centric distance,
* **R** — alarm-count consistency.

These metrics allow the analysis of both drift localization and alarm stability.

## Requirements

The code was developed in Python. The main required packages are:

* numpy,
* scipy,
* scikit-learn,
* matplotlib,
* pandas.

Additional packages may be required depending on the experiment script.

## Reproducibility

The experiments use fixed random seeds to ensure reproducibility. Synthetic streams are generated with predefined parameters, including the number of chunks, chunk size, number of features, number of informative features, number of drifts, and random seed.
